# Product

## Register

product

## Users

From individual users managing their personal documents to B2B organizations using the platform to cross-verify information. Individuals need a simple, private vault for identity docs (PAN, Aadhaar, Voter ID, education certificates); B2B users need reliable data extraction and cross-verification against submitted documents.

## Product Purpose

Store once. Understand forever. Reuse everywhere. Replace the mental overhead of file cabinets and folder structures with an AI that understands the *content* of documents — extracting identity facts, relationships, and verified data that can be reused for autofill, verification, and form submission.

## Brand Personality

Trustworthy, Fast, Modern.

Voice: confident but quiet. No hype, no jargon. The app lets the user's documents speak for themselves.

## Anti-references

- **AI-generated look**: No gradient text, no glassmorphism-as-decoration, no side-stripe accent borders, no purple/cyan dark-mode palettes, no "AI slop" tells. The app should feel engineered, not generated.
- **Enterprise barnacles**: No heavy data tables, crowded dashboards, or the visual language of legacy enterprise software.
- **Traditional file managers**: Not Google Drive, not Dropbox, not a folder tree. Documents are organized by meaning, not by path.

## Design Principles

1. **Identity first, files second** — The UI should surface people, facts, and relationships before folders and filenames.
2. **Confidence through clarity** — Every data point shows how confident the AI is. Ambiguity is surfaced honestly, not hidden.

4. **Privacy as a feature** — The design should feel secure and local even when cloud features exist. No unnecessary data requests, transparent data handling.
5. **One tap, not three** — The primary action on every screen should be reachable without buried menus.

## Accessibility & Inclusion

- WCAG AA compliance target (4.5:1 body text contrast, 3:1 large text)
- Touch targets ≥ 48px
- Screen reader labels on all interactive elements
- Reduced motion: animations should not gate content visibility
