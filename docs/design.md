# Personal AI Vault - Design System

## Design Philosophy

Inspired by Apple.

The product should feel:

* Calm
* Intelligent
* Private
* Premium
* Invisible

The user should interact with knowledge, not files.

---

## Design Principle

Most document apps start with files.

We start with identity.

The UI should prioritize:

People → Facts → Documents

Not:

Folders → Files → Metadata

## Design Goal

The application should feel like a personal memory system.

Users should gradually stop thinking about documents and start thinking about knowledge.

The vault stores files.

The AI understands them.

The identity graph becomes the product.
